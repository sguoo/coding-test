### 문제 설명

<p>Маленький Джек решил написать самую страшную историю, чтобы напугать своих друзей на Хэллоуин.</p>

<p>Назовем историей непустую последовательность из слов, разделенных пробелами. Слово в истории --- непустая последовательность строчных букв латинского алфавита.</p>

<p>Как известно, на качество истории влияют не только слова, содержащиеся в ней, но и символы, содержащиеся в этих словах.</p>

Для проверки каждой гипотезы ему необходимо по номеру символа в истории узнать порядковый номер слова и позицию символа в этом слове.</p>
